# Student Management System

A web-based Student Management System built using Spring Boot, Thymeleaf, Spring Data JPA, Hibernate, and MySQL.

## Features

* Add New Student
* View Student Details
* Update Student Information
* Delete Student Records
* Form Validation
* Responsive User Interface using Bootstrap
* MySQL Database Integration

## Technologies Used

* Java 21
* Spring Boot
* Spring Data JPA
* Hibernate
* Thymeleaf
* MySQL
* Bootstrap 5
* Maven
* Git & GitHub

## Project Architecture

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* DTO Layer
* Thymeleaf Views

## How to Run

1. Clone the repository
2. Configure MySQL in `application.properties`
3. Create database `sms`
4. Run the Spring Boot application
5. Open:

http://localhost:8080/students
